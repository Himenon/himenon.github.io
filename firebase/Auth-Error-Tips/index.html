FirebaseのAuthで躓いたこと¶

signInAndRetrieveDataWithCredential failed: First argument "credential" must be a valid credential. Dismiss¶

状況¶

FirebaseのサンプルコードにFirebase UIをツッコんで検証しようとした。 すると、画像のようなエラーがでた。

原因¶

これ、すごい単純で、firebaseのバージョンが古かったです。

Before

"firebase": "^4.5.0",
"firebaseui": "^2.7.0",

After

"firebase": "^4.12.1",
"firebaseui": "^2.7.0",

対策¶

他のpackageでも言えますが、

  1. Release Noteを見る。
  2. 上記がない場合、package.jsonからfirebase-uiのdependenciesで読み込まれているfirebaseのバージョンを確認する。

package.json#54

2つ目が確実か。
