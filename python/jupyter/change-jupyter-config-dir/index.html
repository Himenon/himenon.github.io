jupyter_notebook_config.pyの読み込み先を変更する

環境変数のJUPYTER_CONFIG_DIRに参照先のディレクトリパスを記述すればよい。

JUPYTER_CONFIG_DIR=/config

Dockerで立ち上げる時に、~/homeを探すのは面倒なので。
