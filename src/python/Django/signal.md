# DjangoのSignalの使い方

* 作成日：2017年11月06日
* 公開日：2018年09月15日

いわゆるイベントフック


```python
@receiver(pre_save, sender=JobOffer)
def before_save_handler(sender, instance, **kwargs):
    print("保存前に呼び出されている")


@receiver(post_save, sender=JobOffer)
def after_save_handler(sender, instance, **kwargs):
    print("保存後に呼び出されれる")
```
