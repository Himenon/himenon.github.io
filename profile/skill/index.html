Julia¶

NodeJS¶

Cloud Functionで利用。 WEBアプリケーションをこれで作成したことはない。 ツールとして利用することは度々あるので、ちょっとしたことはかける。 Headless Chromeを使ってPDF作成など、マイクロサービスとして利用すれば面白そう、という予感はある。

いまはZappierやIFTTTとあそぶ。

  • React
  • Ultradom(旧picodom)
